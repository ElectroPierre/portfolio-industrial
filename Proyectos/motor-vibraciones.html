Proyecto industrial

Monitorización de motor industrial

Sistema para medir vibraciones mediante sensores IMU, procesar datos y visualizar el estado del motor en un dashboard.

ESP32 IMU MQTT Python Grafana

Problema industrial

En muchos entornos industriales, los motores eléctricos pueden presentar vibraciones anómalas por desalineación, desgaste mecánico o fallos en rodamientos.

Este proyecto propone una solución de bajo coste para supervisar vibraciones y detectar posibles comportamientos anómalos.

Arquitectura del sistema

Sensor IMU → ESP32 → MQTT → Base de datos → Dashboard Grafana.
