Proyecto IIoT / Datos

Mini SCADA industrial

Sistema básico de adquisición de datos, almacenamiento, alarmas y visualización para procesos industriales.

Node-RED MQTT PostgreSQL Grafana SQL

Problema industrial

En una planta industrial es importante registrar variables del proceso, visualizar tendencias y detectar alarmas antes de que afecten a la producción.

Este proyecto simula una solución SCADA ligera usando herramientas abiertas para supervisión y análisis de datos industriales.

Arquitectura del sistema

Dispositivo/sensor → MQTT → Node-RED → PostgreSQL → Grafana.

Estado del proyecto

Proyecto orientado a demostrar conocimientos en adquisición de datos, bases de datos, dashboards industriales y alarmas.
