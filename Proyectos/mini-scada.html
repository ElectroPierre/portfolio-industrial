Proyecto IIoT / SCADA / Datos industriales

Mini SCADA Industrial IoT

Sistema IIoT funcional con sensor físico DHT22, ESP8266, comunicación MQTT, almacenamiento en InfluxDB y dashboard industrial en Grafana.

ESP8266 DHT22 MQTT Node-RED InfluxDB Grafana

Resumen del proyecto

Este proyecto implementa un Mini SCADA industrial 100% virtual para supervisar una máquina simulada en tiempo real. El sistema genera variables de proceso, detecta alarmas, almacena históricos y muestra el estado de la instalación mediante un dashboard industrial.

Problema industrial

En una planta industrial es fundamental registrar variables del proceso, visualizar tendencias y detectar condiciones anómalas antes de que afecten a la producción, seguridad o mantenimiento.

Esta solución demuestra cómo implementar una arquitectura ligera y escalable de supervisión usando herramientas abiertas.

Arquitectura del sistema

DHT22 → ESP8266 → WiFi → MQTT Mosquitto → Node-RED → InfluxDB → Grafana

Integración con sensor físico

El sistema incorpora un sensor DHT22 conectado a un ESP8266 NodeMCU. El dispositivo mide temperatura y humedad reales, publica los datos por MQTT y los envía al flujo de Node-RED para almacenarlos en InfluxDB y visualizarlos en Grafana.

Variables monitorizadas

Temperatura

Variable principal para control ambiental y alarmas térmicas.

Humedad relativa

Seguimiento ambiental para salas técnicas o zonas de producción.

Estado máquina

Representación ON/OFF de la línea o activo supervisado.

Alarma técnica

Activación automática por sobretemperatura.

Funciones implementadas

  • Lectura real de temperatura y humedad con DHT22
  • Publicación MQTT desde ESP8266
  • Procesamiento de datos en Node-RED
  • Histórico en InfluxDB
  • Dashboard SCADA en Grafana
  • Alarmas visuales por sobretemperatura

Resultados obtenidos

  • Monitorización en tiempo real de variables de proceso.
  • Histórico de temperatura y humedad.
  • Estado de máquina visible mediante panel tipo semáforo.
  • Alarma técnica con mapeo visual OK/ALARMA.
  • Arquitectura IIoT escalable para futuras máquinas o sensores reales.
