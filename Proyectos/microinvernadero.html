Proyecto de automatización

Microinvernadero automatizado

Sistema de control para temperatura, humedad y ventilación usando PLC, sensores y visualización de datos.

PLC Siemens TIA Portal HMI Sensores Control ON/OFF

Problema industrial

Muchos sistemas agrícolas o de laboratorio necesitan controlar variables ambientales como temperatura, humedad e iluminación.

Este proyecto plantea un sistema compacto para automatizar un microinvernadero y supervisar sus condiciones de funcionamiento.

Arquitectura del sistema

Sensores ambientales → PLC Siemens → Lógica de control → Actuadores → HMI/Dashboard.

Estado del proyecto

Proyecto planteado como prototipo real. Se añadirán esquemas eléctricos, programa del PLC, capturas de HMI y resultados.
